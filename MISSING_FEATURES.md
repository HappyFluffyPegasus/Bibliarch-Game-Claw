# MISSING FEATURES - User Stories

## 1. Character Relationship System
**As a story creator**, I want to define relationships between characters (friends, enemies, lovers, family) so that I can track how they interact and how relationships evolve over time.

**As a player in Life Mode**, I want to see relationship status between characters so that I understand their dynamics.

## 2. Inventory/Item System
**As a story creator**, I want to create items that characters can own and use so that objects can drive plot points (finding a key, using a tool, giving a gift).

**As a player**, I want characters to automatically use items from their inventory to fulfill needs so that the world feels more realistic.

## 3. Visual World Map
**As a story creator**, I want to see a visual map of all my locations so that I can understand the geography and layout of my world.

**As a player**, I want to see where characters are on a map so that I can follow their movements.

## 4. Toast Notification System
**As a user**, I want non-intrusive toast notifications when events happen so that I stay informed without being interrupted.

## 5. Global Search
**As a user with a large project**, I want to search across all stories, characters, locations, and dialogue so that I can quickly find what I'm looking for.

## 6. Recent Files/Quick Access
**As a user**, I want to see my recently opened stories so that I can quickly get back to work.

## 7. Keyboard Shortcuts Reference
**As a power user**, I want to see all available keyboard shortcuts so that I can work more efficiently.

## 8. Undo/Redo System
**As a user**, I want to undo and redo my actions so that I can recover from mistakes.

## 9. Auto-Save
**As a user**, I want my work to be automatically saved so that I don't lose progress if something goes wrong.

## 10. Data Export/Backup
**As a user**, I want to export all my data as JSON so that I can create backups or migrate to another system.

## 11. Character Mood/Status Display
**As a player**, I want to see a character's current mood and status so that I understand their emotional state.

## 12. Event History/Timeline
**As a story creator**, I want to see a history of all events that have occurred in Life Mode so that I can review what happened.
